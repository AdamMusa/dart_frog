A fast, minimalistic backend framework for Dart

## Quick Start 🚀

```sh
# 📦 Install the dart_frog cli
dart pub global activate --source path ./packages/dart_frog_cli

# 🧑‍💻 Run the example dev server
cd ./example
dart_frog dev
```
